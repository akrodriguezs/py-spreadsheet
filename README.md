

# 🧮 Aplicación de Excel

Aplicación de escritorio desarrollada en Python utilizando CustomTkinter para simular funcionalidades básicas de una hoja de cálculo.

---

## ✨ Características

- Interfaz gráfica moderna
- Edición de celdas
- Soporte para fórmulas básicas
- Evaluación dinámica de datos

---

## 📦 Instalación y ejecución

Sigue estos pasos para ejecutar el proyecto en tu computadora.

### 1️⃣ Clonar el repositorio
```bash
git clone https://github.com/akrodriguezs/py-spreadsheet.git
cd py-spreadsheet


---

### 2️⃣ Crear un entorno virtual

```bash
python -m venv venv
```

---

### 3️⃣ Activar el entorno virtual

**En Windows:**

```bash
venv\Scripts\activate
```

**En Linux / Mac:**

```bash
source venv/bin/activate
```

---

### 4️⃣ Instalar dependencias

```bash
pip install -r requirements.txt
```

---

### 5️⃣ Ejecutar la aplicación

```bash
python main.py
```

---

## 🧠 Requisitos

* Python 3.8 o superior
* pip instalado

---

## ⚠️ Notas

* Asegúrate de tener `tkinter` instalado (generalmente ya viene con Python).
* Si ocurre algún error con dependencias, intenta actualizar pip:

```bash
pip install --upgrade pip
```
